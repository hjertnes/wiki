C-x #

  • Emacs
  • Emacs Writing

If you use Emacs as your $EDITOR you need to know C-x #. This is the shortcut that will communicate back to the terminal that you're done editing.

For those who don't know: $EDITOR is for the env variable called EDITOR that all unix like (and windows at this point) operating systems have defined. It points at your preferred editor or usually nano if you don't do anything.
