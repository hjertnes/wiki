Where emacs doesn't work for me

  • Notes
  • Emacs
  • Emacs Writing

I use emacs for a lot of stuff. But there are some stuff where it simply put doesn't work for me.

  • Software development: IDE's are simply put a lot better for most serious software development
  • E-mail: the setup never works for me, and turns out I don't do that much e-mail really.
  • RSS: having a GUI works way better

Where emacs works for me is mostly stuff where I write a lot.
