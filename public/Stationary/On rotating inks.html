I think most fountain pen people doesn't do it like this. But I'm weird.

  • I use the same ink in all of my pens
  • I don't change inks a lot
  • Usually when the bottle is empty
  • But not before I've used at least 1/3 of it
  • If I don't like a ink, I throw it out.
