Lingon X is a great app that let you manage Launchd.

Launchd is what have replaced cron on modern version of macOS. It is what you want if you are lookig for something to run on a given interval.

Cron had a very simple format, that most technical users can manage without too much fuss. Launchd on the other hand has this complex xml format. So you really want an app.
